# Customer Behavior Analysis and Visualization

## Project Overview

This project analyzes customer shopping behavior to identify patterns, trends, and business insights. The analysis was performed using Python for data processing and exploratory data analysis, SQLAlchemy for database integration, Power BI for dashboard visualization, and Streamlit for live web application deployment.

The goal of this project is to understand customer demographics, spending behavior, and product trends to support data-driven business decisions.

## Objectives

- Understand customer demographics and purchase behavior  
- Identify high-value customers and spending patterns  
- Analyze product category popularity  
- Perform exploratory data analysis using Python  
- Create business dashboard using Power BI  
- Deploy interactive web application using Streamlit  


## Dataset Information

The dataset contains customer shopping records with features such as:

- Customer ID
- Age
- Gender
- Item Purchased
- Category
- Purchase Amount (USD)
- Review Rating
- Subscription Status
- Shipping Type
- Discount Applied
- Promo Code Used
- Previous Purchases
- Payment Method
- Frequency of Purchases
